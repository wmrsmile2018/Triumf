ReactJs project

## Clone Project

```
git clone https://github.com/wmrsmile2018/Triumf.git
```

## Installation:

```
cd Triumf/
npm install
```

## Run Project

Project work at ```localhost:8080```

via npm start
```
cd Triumf/
npm start
```

via npm run build
```
cd Triumf/
npm run build
npm install -g serve
serve -l 8080 -s dist
```
